Concurrent chatroom with private messaging ability. Also creates a logfile using file i/o. Utilized pthread library for thread operations.

Completed as group effort with

 * 	Sivaramharesh Siva

Instructions:

1. Use the makefile
2. To start the server type in the following: 
	"./chatserver"
3. To connect with client type in the following:
	"./chatroom [ip address]"
